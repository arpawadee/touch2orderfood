	2.1 ดาวน์โหลดโฟลเดอร์ t2orderfood.sql ในโฟลเดอร์ Database

3. การติดตั้ง mobile application บน Windows

	3.2 เปิดโฟลเดอร์ T2OrderFood ด้วย Android Studio 
	3.3 หากรันด้วยโทรศัพท์จริง ให้แก้ไข IP Address ในไฟล์ Config.java ด้วย IP Address ของท่าน 


4. การติดตั้ง Web application บน Windows

	4.2 นำโฟลเดอร์ t2orderfood ที่ดาวน์โหลดสักครู่ ไปวางในโฟลเดอร์ xampp\htdocs
